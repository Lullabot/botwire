Botwire is designed to be a boilerplate for building modular HTML wireframes using Jekyll for the modularity of HTML partials, Sass for CSS partials, and optionally other pre-baked solutions like Bourbon and Bourbon Bitters for some ready-made styles and mixins, Bourbon Neat for the grid system, and optionally Foundation for some pre-baked components and styles.
